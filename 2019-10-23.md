## 基本概念
    实际，列表和元组是一个可以放置任意数据类型的集合。
    而在大多数语言中集合所包含的数据类型必须一致，，而python中列表和元组则不同。他们可以包含任意数据类型的数据
```python
l = [1,'first',2]  #列表可以同时包含int、string类型
>>> l
[1, 'first', 2]
tup1 = (1,'first')  #元组可以同时包含int、string类型
>>> tup1
(1, 'first')
```
